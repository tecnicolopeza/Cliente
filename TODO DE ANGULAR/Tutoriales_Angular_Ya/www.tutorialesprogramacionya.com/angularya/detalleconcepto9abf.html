56 - Otras librerías de componentes visuales que compiten con Angular Material

Si bien en algunas situaciones esta bien evitar el uso de bibliotecas externas durante el desarrollo de aplicaciones Angular, a veces, pueden ayudar a reducir significativamente el tiempo de desarrollo de la aplicación.

No es una exigencia que debamos utilizar obligatoriamente la librería de Angular Material para la implementación de un proyecto, como veremos más adelante muchos desarrolladores optan por utilizar directamente el framework Bootstrap o una versión modificada de Bootstrap adaptada para angular NG Bootstrap.

También es importante hacer notar que las dos librerías nombradas (Angular Material y NG Bootstrap) no son las únicas librerías de componentes visuales disponibles, sino que hay decenas y muchas de ellas las podemos consultar desde el mismo sitio oficial de Angular en la sección de recursos de Angular.

Hay que tener en cuenta también que algunas de estas librerías tienen un costo para su uso.


Los enlaces a las librerías más utilizadas de componentes visuales para Angular son:

  • Angular Material

  • ng-bootstrap

  • ngx-bootstrap

  • PRIMENG

  • ag-Grid

  • Sencha for Angular

  • Onsen UI

  • Clarity Design System
